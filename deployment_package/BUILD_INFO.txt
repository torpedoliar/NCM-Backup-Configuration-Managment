  TROUBLESHOOTING QUICK REFERENCE
═══════════════════════════════════════════════════════════════

ISSUE: Schedules not running after GUI closed
FIX:   Read SOLUTION_SCHEDULE_TIDAK_JALAN.txt
       Must setup auto-start (one-time only)

ISSUE: Schedule enabled but not executing
FIX:   Check logs/app.log for errors
       Verify service status = "Running"

ISSUE: Service status "Not Running"
FIX:   Settings → "Setup Auto-Start"
       OR Settings → "Start Service Now"

ISSUE: "Master passphrase file not found" in logs
FIX:   Setup auto-start never done
       Settings → "Setup Auto-Start"

ISSUE: "NO ENABLED SCHEDULES FOUND" in logs
FIX:   Enable schedules in Schedules tab
       Check "Enabled" column shows ✓
