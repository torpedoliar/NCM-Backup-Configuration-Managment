═══════════════════════════════════════════════════════════════

2. Run Allied_Telesis_Backup_Manager_v3.0.exe

═══════════════════════════════════════════════════════════════
  TROUBLESHOOTING
═══════════════════════════════════════════════════════════════

Issue: Application won't start
Fix: Check logs/app.log for errors
     Run as Administrator if needed

Issue: Can't connect to switches
Fix: Verify network connectivity (ping)
     Check firewall settings
     Verify credentials in Credentials tab

Issue: Schedules not running after GUI closed
Fix: Check Task Scheduler - "AlliedTelesisBackup" should run
     Check logs for [BACKGROUND-JOB] entries
     Verify service auto-start is configured

Issue: Forgot master password
Fix: Delete data/app.db (WARNING: loses all data)
     Or use Factory Reset in Settings tab
